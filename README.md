# author
Author home page for Anonymous Ryder

This is a clean, simple homepage for an author, featuring a bio and contact information.

## Getting Started

To view the homepage, run the local server:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Files

- `index.html`: The main homepage HTML.
- `style.css`: Clean CSS styling.
- `script.js`: Basic JavaScript for the page.
